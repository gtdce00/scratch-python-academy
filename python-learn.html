1. การพิมพ์ข้อความออกหน้าจอ (print)

ใน Scratch

พูด สวัสดี!

ใน Python

print("Hello!")

คำแนะนำและไวยากรณ์

ในโปรแกรม Scratch น้องๆ จะใช้บล็อกสีม่วง พูด (say) เพื่อแสดงกล่องข้อความจากตัวละครบนเวที แต่ในภาษา Python คำสั่งที่ตรงกันคือฟังก์ชัน print() ซึ่งจะพิมพ์ข้อความที่ต้องการลงไปในคอนโซล (Terminal)

🎯 กิจกรรมท้าทาย:

  • เขียนคำสั่งพิมพ์คำตอบ print("Hello Python") ลงในช่อง Editor ด้านขวา
  • กดปุ่ม "รันโค้ด Python" เพื่อวิเคราะห์คำสั่ง
