โปรแกรม Scratch คืออะไร ?

Scratch เป็นโปรแกรมสร้างสื่อมัลติมีเดียที่ใช้งานได้ทั้งแบบออฟไลน์ (Offline) และออนไลน์ (Online)

เหมาะอย่างยิ่งสำหรับนักเรียน นักวิชาการ ครู และผู้ปกครองที่ต้องการสร้างชิ้นงานการศึกษา หรือเพื่อความบันเทิง เช่น:

  • โครงงานคณิตศาสตร์และวิทยาศาสตร์
  • แบบจำลองและสถานการณ์จำลองเพื่อการเรียนรู้
  • งานนำเสนอ สื่อเคลื่อนไหว เล่าเรื่องราว (Animation)
  • งานด้านศิลปะ ดนตรีแบบมีปฏิสัมพันธ์ และเกมส์ง่ายๆ
สวัสดีคร้าบ! ผมเหมียวส้ม ยินดีที่ได้รู้จักครับ!

โครงสร้างสองขุมพลังของ Scratch

โปรแกรม Scratch ประกอบไปด้วย 2 ส่วนสำคัญ ได้แก่:

1. โปรแกรมคอมพิวเตอร์

มีหน้าที่ในการสร้าง จัดการภาพ เสียง และสคริปต์ (Script) ในชิ้นงานของเรา

2. ภาษาคอมพิวเตอร์ (ภาษา Scratch)

ทำหน้าที่ผสาน ภาพ เสียง สร้างกระบวนการ และการโต้ตอบกับผู้ใช้งาน

ทำไมต้องเป็น "การเขียนโปรแกรมแบบบล็อก" (Block Programming)?

การเขียนโปรแกรมใน Scratch มีความแตกต่างจากภาษาคอมพิวเตอร์ทั่วไป คือ "เราใช้วิธีลากบล็อกคำสั่งมาเชื่อมต่อกันแทนการพิมพ์โค้ด" ซึ่งมีข้อดีดังนี้:

มองเห็นขั้นตอนการทำงานเป็นภาพ เข้าใจง่ายทันที
ลดความผิดพลาดจากการสะกดคำหรือพิมพ์ไวยากรณ์ผิด
บล็อกมีรูปร่างเฉพาะที่ถูกออกแบบให้ต่อเข้ากันได้เฉพาะคู่ที่ถูกต้องเท่านั้น ป้องกันการรันโปรแกรมผิดรูปแบบ

เครื่องมือที่จำเป็นในการออกแบบและพัฒนา Scratch

ก่อนเริ่มต้นพัฒนาชิ้นงานจริง นักเขียนโปรแกรมจะใช้เครื่องมือต่างๆ ในการวางแผนและเตรียมงาน ดังนี้:

1. โปรแกรมเขียนผังงาน (Flowchart)

เช่น Microsoft Visio หรือ Microsoft Word ใช้จัดเรียงลำดับความคิดอย่างเป็นขั้นเป็นตอน

2. โปรแกรมออกแบบหน้าจอ

เช่น Microsoft PowerPoint ใช้วางองค์ประกอบของหน้าจอ วาดภาพร่างตัวละครและฉากหลัง

3. โปรแกรมพัฒนาระบบ

โปรแกรม Scratch Version 3.0 สำหรับการเขียนโปรแกรมสั่งการ

ตัวดำเนินการ (Operators)

ตัวดำเนินการจะถูกนำไปใช้งานร่วมกับตัวแปรและค่าคงที่ เพื่อกระทำสิ่งต่างๆ เช่น การคูณเลข การเปรียบเทียบค่า หรือตรวจสอบความถูกต้อง โดยแบ่งออกเป็น 3 ประเภทหลัก:

ตัวดำเนินการคณิตศาสตร์ (Arithmetic Operators)

ใช้คำนวณตัวเลขระหว่างสองค่าเพื่อหาผลลัพธ์ทางคณิตศาสตร์

สัญลักษณ์ ชื่อตัวดำเนินการ ตัวอย่าง คำอธิบาย
+ การบวก (Addition) c = a + b นำค่าของ a และ b มารวมกัน
- การลบ (Subtraction) c = a - b หักออกด้วยค่าของ b จากค่า a
* การคูณ (Multiplication) c = a * b คูณค่าของ a ด้วยค่าของ b
/ การหาร (Division) c = a / b หารค่าของ a ด้วยค่าของ b
% การหารเอาเศษ (Modulo) c = a % b หารจำนวนเต็มแล้วเอาเศษที่เหลือจากการหาร (เช่น 5 % 2 ได้ 1)

ตัวดำเนินการเปรียบเทียบ (Relational & Comparison Operators)

ใช้ประเมินค่าระหว่างสองค่าเพื่อหาผลลัพธ์ว่าเป็น จริง (True) หรือ เท็จ (False)

สัญลักษณ์ ตัวอย่าง ผลลัพธ์ที่จะเป็นจริง (True)
== a == b เมื่อค่าของ a เท่ากับ ค่าของ b
!= a != b เมื่อค่าของ a ไม่เท่ากับ ค่าของ b
< a < b เมื่อค่าของ a น้อยกว่า ค่าของ b
> a > b เมื่อค่าของ a มากกว่า ค่าของ b
<= a <= b เมื่อค่าของ a น้อยกว่าหรือเท่ากับ ค่าของ b
>= a >= b เมื่อค่าของ a มากกว่าหรือเท่ากับ ค่าของ b

ตัวดำเนินการทางตรรกศาสตร์ (Logical Operators)

ใช้ผสานหลายเงื่อนไขย่อยให้เหลือผลลัพธ์เดียว โดยปลายทางสุดท้ายจะเป็นจริงหรือเท็จ

ชื่อตรรกะ สัญลักษณ์ ตัวอย่าง คำอธิบายแบบเข้าใจง่าย
not (ไม่) ! !a จะกลับผลลัพธ์เป็นตรงกันข้าม จากจริงเป็นเท็จ จากเท็จเป็นจริง
and (และ) && a && b จะเป็นจริงได้เมื่อ ทั้งเงื่อนไข a และ b ต้องเป็นจริงพร้อมกัน เท่านั้น
or (หรือ) || a || b จะเป็นจริงได้เพียงขอแค่ มีเงื่อนไขใดเงื่อนไขหนึ่งเป็นจริง ก็ผ่านฉลุย

ผังงาน (Flowchart) คืออะไร?

"ภาพหนึ่งภาพ แทนคำอธิบายได้เป็นล้านคำ" นั่นคือหัวใจของผังงาน!

ผังงาน (Flowchart) คือ แผนภาพแสดงการทำงานของโปรแกรม โดยใช้ สัญลักษณ์สากล เพื่ออธิบายขั้นตอนและลักษณะการทำงานในแต่ละจุด สัญลักษณ์เหล่านี้จะเชื่อมโยงกันด้วยลูกศร เพื่อบอกทิศทางการเคลื่อนที่ของโปรแกรม ช่วยให้เรา:

ประโยชน์ของการเขียนผังงาน

  • เรียนรู้และเข้าใจขั้นตอนได้รวดเร็ว (ไม่ต้องติดข้อจำกัดด้านภาษาโปรแกรมมิ่ง)
  • มองเห็นภาพลำดับการทำงานและทิศทางข้อมูลรวมของทั้งระบบชัดเจน
  • ค้นหาข้อผิดพลาด (บั๊ก) ได้เร็ว และซ่อมแซมแก้ไขโปรแกรมได้ง่าย
  • ช่วยให้เขียนโปรแกรมได้รวดเร็วและเป็นระบบมากขึ้นเมื่อทำตามผังงาน

ประเภทของผังงานคอมพิวเตอร์

  1. ผังงานระบบ (System Flowchart): แสดงภาพกว้างๆ ของระบบว่าส่วนสำคัญใดสัมพันธ์กันบ้าง
  2. ผังงานโปรแกรม (Program Flowchart): แสดงรายละเอียดของขั้นตอนคำสั่งและการคำนวณอย่างเป็นลำดับเพื่อนำไปโค้ดดิ้ง

มาฝึกจำสัญลักษณ์ผังงานกันเถอะ!

ก่อนจะไปเริ่มเขียนโปรแกรม มาเล่นเกมจับคู่รูปทรงสัญลักษณ์และคำอธิบายหน้าที่ของมันกันเถอะ วิธีการเล่นคือ คลิกเลือกรูปสัญลักษณ์ด้านซ้าย แล้วคลิกเลือกคำอธิบายขวาที่คู่กัน ให้ครบทุกข้อนะ!

สำรวจส่วนประกอบห้องควบคุม

กรุณากดคลิกปุ่มตัวเลข 1 ถึง 8 ในหน้าจอโปรแกรมจำลองด้านข้าง เพื่อดูคำอธิบายของแถบเครื่องมือแต่ละส่วน

ทำความรู้จัก 8 หมวดหมู่บล็อกคำสั่ง (Scripts Categories)

บล็อกคำสั่งของ Scratch ถูกจัดหมวดหมู่ตามสีต่างๆ เพื่อช่วยให้เราจำได้ง่ายและสลับเครื่องมือได้รวดเร็ว:

Motion (เคลื่อนไหว) ใช้สั่งเคลื่อนตัวละครเดินหน้า ถอยหลัง หมุน หมุนไปตำแหน่งต่างๆ
Looks (รูปลักษณ์) สั่งพูด คิด เปลี่ยนชุดตัวละคร ปรับสี/ขนาด ซ่อนหรือแสดงตัวละคร
Sound (เสียง) เล่นเสียงเอฟเฟกต์ เพิ่มความดัง ปรับระดับเสียงคีย์เปียโนหรือดนตรี
Events (เหตุการณ์) กำหนดจุดเริ่มต้นการรันโปรแกรม เช่น เมื่อคลิกธงเขียว หรือกดคีย์บอร์ด
Control (ควบคุม) สั่งรันวนซ้ำตลอดไป วนตามจำนวนรอบ เช็คเงื่อนไข (If-Else) สั่งหยุด
Sensing (ตรวจจับ) ตรวจจับการชนขอบเวที ตัวละครแตะสี วัดพิกัดเมาส์ จับเวลาการรอคอย
Operators (ดำเนินการ) บล็อกสีเขียวคำนวณ บวก ลบ คูณ หาร สุ่มเลข เปรียบเทียบค่าความจริง
Variables (ตัวแปร) สร้างกล่องเก็บข้อมูล กำหนดค่าเริ่ม (Set) และการบวกเพิ่มลดค่า (Change)
