my variable (บล็อกรายงานข้อมูล)

บล็อกวงรีรายงานค่า ใช้ดึงค่าที่ตัวแปรตัวนั้นๆ จดจำเก็บเอาไว้ในกล่องไปคำนวณส่งผลลัพธ์ต่อ

ตั้งค่า my variable เป็น 0

set variable to (ตั้งค่าตัวแปร)

สั่งทับค่าเดิมในตัวแปรตัวนั้นให้เปลี่ยนไปเท่ากับค่าตัวเลขหรือข้อความตัวใหม่โดยตรง

เปลี่ยน my variable ทีละ 1

change variable by (เปลี่ยนค่าสะสมตัวแปร)

สั่งเพิ่มค่าสะสมให้ตัวแปร (ป้อนค่าบวก) หรือลดค่าสะสมลง (ป้อนค่าลบ) เช่น การเพิ่มคะแนนสะสม

แสดงตัวแปร my variable

show variable (แสดงตัวแปรบนเวที)

สั่งให้เปิดการแสดงผลพาเนลบอกสถานะค่าตัวแปรตัวนั้นขึ้นมาบนหน้าต่างหน้าจอเวทีหลัก

ซ่อนตัวแปร my variable

hide variable (ซ่อนตัวแปรบนเวที)

สั่งปิดซ่อนพาเนลแสดงผลค่าตัวแปร ไม่ให้โชว์บดบังตัวละครบนพื้นที่ของเวที

โปรเจกต์มินิต้นแบบ 5 สไตล์เขียนเล่นใน Scratch

น้องๆ สามารถเลือกโปรเจกต์ที่ต้องการศึกษาจากแถบเมนูด้านซ้าย เพื่อดูข้อมูลวัตถุประสงค์ ขั้นตอนความคิด (Flowchart) ชุดบล็อกที่ใช้ และยังกดรันโปรแกรมจำลองเพื่อดูการแสดงผลเสมือนของโปรเจกต์จริงบนเวทีได้เลยครับ!
